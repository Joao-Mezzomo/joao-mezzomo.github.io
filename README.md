Última atividade avaliativa da matéria Criação de Página Web do Curso de Ciência da Computação do Instituto Federal Sul-rio grandense Campus Passo Fundo.
Esse projeto foi combinado através do Plano de Ensino referente ao período 2025/2, sendo o docente da disciplina o Professor Leonardo Deliyannis Constantin.
Nesse projeto, em nosso caso estruturado em trio, utilizamos a tecnologia Git e Github para aprender sobre versionamento e compartilhamento do projeto, afim de criar uma página web responsiva, estilizada e com conteúdo que interessa os integrantes do grupo e terceiros.
Dividimos as funções de maneira que:
1) João Mezzomo é o principal responsável pelo HTML, adicionando todo o conteúdo refente a campanha de RPG do mesmo.
2) Arthur Balke é o principal responsável pelo CSS, adaptando a página para que ela fique dentro da estilização da obra de inspiração do RPG, e também a organização do repositório seguindo a estrutura de pastas corretamente. 
3) Paulo Augusto é o principal responsável pelo JavaScript, implementando novas funcionalidades e responsividades dentro do projeto.
